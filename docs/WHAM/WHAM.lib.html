WHAM.lib.timeseries module

Defines functions for dealing with correlated timeseries data.

WHAM.lib.timeseries.bootstrap_independent_sample()

Draws an independent sample of size N_ind = N/g from x. If the statistical inefficiency g is not passed as a parameter, it will be calculated first.

Parameters

  • x (ndarray) – 1-dimensional array of length N containing correlated data to draw (uncorrelated) sample from.

  • g (float) – Statistical inefficiency (default=None).

Returns

1-dimensional array of length N/g containing random samples drawn from x (with replacement).

Return type

y (ndarray)

WHAM.lib.timeseries.bootstrap_window_samples()

Draws a bootstrap sample from each window.

Parameters

  • x_it (list) – Nested list of length S, x_it[i] is an array containing timeseries data from the i’th window.

  • g (float) – Statistical inefficiency (default=None).

Returns

Nested list of length S, x_it_b[i] is a 1-dimensional array of

length N/g containing random samples drawn from x_it[i] (with replacement).

Return type

x_it_b (list)
